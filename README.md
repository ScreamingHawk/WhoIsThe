# WhoIsThe...

A game for picking *Who Is The ...* out of your friends.

## Development

### Set Up

Install dependencies

```sh
yarn && cd client && yarn
```

### Start

Single command, this enables *watching*

```sh
yarn run dev
```

### Test

```sh
yarn test
```

### Deploy

Set up heroku

```sh
heroku login
```

Do the deployment

```sh
git push heroku master
```
